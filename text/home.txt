Thank you for accessing the Turneffe Reef-Monitoring Data Reporting Tool. This tool is intended for use by Turneffe Atoll Sustainability Association (TASA) to create standardized reports from reef monitoring data, increasing the efficiency of in-house reporting.

If you notice any errors or issues with the app, please submit a bug report using the Submit Bug Report button.
Identify the data you would like to report on: one or more datafiles for a supported reef monitoring datatype.
Ensure each datafile is formatted according to the template on the Data Templates tab for that datatype. 
Upload the datafiles on the corresponding Create Reports tab for that datatype.
Press the Validate Data button to perform a data validation operation on the upload. 
Enter your name and press Download Report. 
For additional information on using and maintaining this app, please visit the Manual tab.
Our data validation checks your submitted data for completeness, standardization, and possible errors. Please note validation works on entire files only, not chosen period/year subsets. 
Please upload only .xls or .xlsx datafiles. If you are unsure how to format your data, please reference the Data Templates panel for guidance. 
Please enter your name or initials, so it may be disclosed in the report who generated the document.
